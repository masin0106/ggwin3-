GitHub Actions Download Studio

GitHub Actions の Ubuntu runner 上で yt-dlp と ffmpeg を実行し、変換後のファイルを 30 分だけ保持する一時ダウンロードページです。VPS 側は制御と表示のみを担当し、ファイル本体は runner 上でのみ保持されます。
